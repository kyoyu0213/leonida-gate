# Leonida Gate - GTA6 Community Hub デザイン案

## 選定デザイアプローチ：サイバーパンク・ネオンコマンド

### デザインムーブメント
**サイバーパンク × テック・ノワール**

80年代のレトロフューチャーと現代的なダークUIを融合させたビジュアル。グリッチ効果、ネオンカラー、デジタルグリッドが特徴。GTA6の未来的な世界観と、情報発信・コミュニティ形成というミッションを体現する。

### コア原則
1. **高コントラスト・ネオン配色** - 深い黒背景に蛍光ピンク・シアン・ライムグリーンが映える
2. **デジタル・グリッド感** - 背景に微細なグリッドパターン、セクション区切りに斜線・グリッチ効果
3. **テック・ノワール・ムード** - 冷たく、エッジが効いた、でも親しみやすい雰囲気
4. **情報階層の明確性** - 重要な情報は大きく、ネオンで強調；補足情報は落ち着いた色

### カラー哲学
- **背景** - `#0a0e27` (深い紫紺) - 夜間のサイバーシティ
- **プライマリネオン** - `#ff006e` (蛍光ピンク) - CTA、重要な見出し、アクセント
- **セカンダリネオン** - `#00f5ff` (シアン) - 情報パネル、ハイライト
- **アクセント** - `#39ff14` (ライムグリーン) - サブ要素、ホバー効果
- **テキスト** - `#e0e6ff` (淡いラベンダー白) - 本文
- **グレー** - `#4a5568` (ダークグレー) - 補足情報、ボーダー

### レイアウトパラダイム
- **非対称・ダイナミック** - 左右で異なる配置、斜線カットで動的感を演出
- **グリッド + オーバーレイ** - 背景にサブトルなグリッド、前景に浮かぶカード
- **セクション分割** - SVG波線ではなく、グリッチ効果やスラッシュラインで区切る
- **スクロール時の視差効果** - ネオンエレメントが背景より速く動く

### シグネチャエレメント
1. **グリッチ枠線** - テキストやカードの周囲に二重線、わずかにずれた効果
2. **ネオンボーダー** - ボタン・カード・入力欄に蛍光ピンク/シアンのボーダー
3. **デジタルグリッド背景** - 全体に透明度低いグリッドパターン

### インタラクション哲学
- **ホバー時のグロー効果** - ボタンやリンク上でネオンが光る
- **クリック時のグリッチ** - わずかな画面ズレ、サウンドエフェクト的な視覚フィードバック
- **スムーズなスライドイン** - セクション出現時に左右からスライド、ネオンが先に光る
- **フォーカス状態の強調** - フォーム要素がアクティブになるとネオン枠が点滅

### アニメーションガイドライン
- **グロー脈動** - `0.8s` の周期で明るさが変わるアニメーション（`opacity`）
- **グリッチ効果** - `0.15s` の短い間隔で `transform: translateX(2px)` を繰り返す（ホバー時のみ）
- **スライドイン** - `0.4s ease-out` で左から右へ、または上から下へ
- **フェード** - `0.3s ease-in-out` で透明度変化
- **ボタンプレス** - `0.1s` で `scale(0.95)` + `filter: brightness(1.2)`

### タイポグラフィシステム
- **見出し（H1/H2）** - `Space Mono Bold` 24-48px、ネオンピンク、文字間隔広め
- **サブ見出し（H3/H4）** - `Space Mono Regular` 16-24px、シアン
- **本文** - `Roboto` 14-16px、淡いラベンダー白
- **キャプション** - `Roboto Mono` 12px、ダークグレー

### ブランドエッセンス
**ワンライン定位置** - 「GTA6の最新情報をリアルタイムで日本語化し、RPサーバーコミュニティの中心地となるプラットフォーム。テック・ノワール美学で、ゲーマーの知的好奇心と創造性を刺激する。」

**3つのパーソナリティ形容詞**
1. **エッジの効いた** - 洗練されたサイバーパンク美学
2. **アクセシブル** - 複雑さの中にも親しみやすさ
3. **ダイナミック** - 常に動き、進化し続ける

### ブランドボイス
- **見出し・CTA** - 短く、命令的、テック用語を混ぜる
  - 例：「**ENTER THE GATE**」「**DECODE GTA6**」「**JOIN THE NETWORK**」
- **本文** - 情報的で親しみやすい、ゲーマー向けの口調
  - 例：「最新のGTA6情報を、日本語で、リアルタイムに。」

### ロゴ・ワードマーク
**ロゴマーク** - 「L」と「G」を組み合わせたサイバーシンボル、ネオンピンク＆シアンの二重線で構成。透明背景PNG、正方形。

### シグネチャブランドカラー
**ネオンピンク `#ff006e`** - Leonida Gateを象徴する唯一無二の色。すべてのCTA、重要な見出し、ブランド要素に使用。

## 実装ガイドライン

### CSS変数（index.css）
```css
--primary: #ff006e;          /* ネオンピンク */
--secondary: #00f5ff;        /* シアン */
--accent: #39ff14;           /* ライムグリーン */
--background: #0a0e27;       /* 深い紫紺 */
--foreground: #e0e6ff;       /* 淡いラベンダー白 */
--border: #4a5568;           /* ダークグレー */
```

### フォント
- Google Fonts: `Space Mono` (見出し), `Roboto` (本文), `Roboto Mono` (コード/キャプション)

### 背景パターン
- SVG グリッドパターン（透明度 5-10%）を全体に配置
- セクション区切りにグリッチ効果またはスラッシュライン

### レスポンシブ
- **モバイル** - 1カラムレイアウト、ネオンボーダー幅を減らす
- **タブレット** - 2カラムグリッド
- **デスクトップ** - 3カラムグリッド + サイドバー情報パネル
